# Installation

## Requirements

- Python 3.11 or higher

## Install from PyPI

```bash
pip install chico-ai
```

## Install with uv

```bash
uv tool install chico-ai
```

## Verify

```bash
chico --help
```

## Environment variables

| Variable | Description |
|---|---|
| `GITHUB_TOKEN` | GitHub personal access token for private repos |
